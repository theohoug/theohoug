### Event Horizon&ensp;—&ensp;[fade.run](https://fade.run)

A scroll-driven WebGL journey into a black hole.

65,000 GPU-driven particles with real-time gravitational physics across 9 chapters of astrophysics. Custom GLSL shaders handle all particle trajectories directly on the GPU — no physics library, every equation written from scratch. Spatial audio reacts to scroll position. Runs at 60fps in Chrome.

No game engine. No 3D software. Just code.

<br>

`Three.js`&ensp;`GLSL`&ensp;`WebGL`&ensp;`Web Audio API`&ensp;`GSAP`
